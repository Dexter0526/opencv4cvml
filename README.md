## 소스 코드 빌드 방법
---
* Windows & Visual Studio 2017
    ```
    * Visual Studio 2017에서 *.sln 파일을 불러온 후 [빌드] -> [솔루션 빌드] 메뉴를 선택하세요.
    * 프로그램 실행은 [디버그] -> [디버그하지 않고 시작] 메뉴를 선택하세요.
    ```

* Linux
    ```bash
    $ cd <project>
    $ cmake .
    $ make
    $ ./project
    ```
